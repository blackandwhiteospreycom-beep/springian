# Project: Springian

## Overview
Springian is an all-in-one business management platform similar to Odoo, Zoho, and Monday.com.
It includes modules like CRM, Project Management, HRM, Finance, and much more. 

## Tech Stack
- Frontend: React + Vite
- Backend: Node.js / Express
- Database: (PostgreSQL)
- API Style: REST

## Architecture
- Modular system (each feature = module)
- Clean separation: frontend / backend / services
- Reusable components and APIs

## Coding Rules
